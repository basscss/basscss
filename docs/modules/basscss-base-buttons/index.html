Base Buttons

NPM Github

Structural styles and resets for buttons are set in this base element styles module. All buttons share common padding and height, well suited for tap targets on touchscreen displays. Color styles in other modules and other custom extensions adjust their surface-level appearance.

Default Variables

To customize base button styles, use the following variables.

:root {
  --button-font-size: inherit;
  --button-font-weight: bold;
  --button-line-height: 1.125rem;
  --button-padding-y: .5rem;
  --button-padding-x: 1rem;
}

Note: adjusting the line-height and padding may require adjustments to form element variables to maintain vertical alignment.
